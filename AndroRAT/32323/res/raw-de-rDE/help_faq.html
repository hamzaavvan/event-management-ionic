Einnahmen durch Werbung

Es existiert die Auffassung, dass wenn ein Nutzer nie auf Werbung klickt, dies einer Webseite oder Anwendung finanziell nicht schaden könnte. Entwickler werden aber oft über die Anzahl der Views bezahlt. Es bleibt deine Entscheidung ob du Werbung blockierst oder nicht. Ich persönlich würde keine Anwendungen verwenden die Werbung anzeigen, weil mich diese äußerst stören. Ohne einem Werbeblocker würde ich diese Anwendungen nicht installieren.

Wie funktioniert AdAway?

AdAway benutzt die Hosts-Datei um Hostnamen zu blockieren, die Werbung ausliefern. Die Hosts-Datei ist eine Datei, die sich in /system/etc/hosts befindet und Hostnamen in IP Adressen übersetzt. Dies ist eine Möglichkeit Hostname-IP-Paare zu definieren ohne vom Domain Name System (DNS) Gebrauch zu machen. Alle unerwünschten Hostnamen werden zu 127.0.0.1 umgeleitet, also auf dein eigenes Gerät.

Warum ist die Einstellung für den täglichen Update-Check ausgegraut?

Automatische Überprüfung für neue Updates ist nur verfügbar wenn AdAway auf dem internen Telefonspeicher installiert ist. Dies ist aufgrund von Beschränkungen von der Android Plattform.

Warum muss ich Android neu starten, damit die Änderungen wirksam werden?

Java auf Android hat einen eigenen internen DNS-Buffer. Das Betriebssystem übernimmt die neue Hosts-Datei sofort (dies kann man mit ping über die Kommandozeile überprüfen). Jedoch ist es nötig Android neuzustarten um Javas DNS-Buffer zu erneuern.

Wie verwendet man den Webserver in AdAway?

AdAway funktioniert auch ohne den Webserver zu aktivieren!

Du kannst den lokalen Webserver in den Einstellungen von AdAway aktivieren, damit Anfragen an die lokale IP-Adresse 127.0.0.1 beantwortet werden. Das bedeutet das Anfragen von Anwendungen an Server, die eigentlich Werbung ausliefern, nun von AdAways Webserver beantwortet werden.
Manche Anwendungen funktionieren nicht wenn Werbeserver nicht erreichbar sind. Mit dieser Methode werden sie wieder erreichbar, da der lokale Webserver mit einer leeren Seite antwortet.

Wie kann ich bestimmte Hostnamen sperren/entsperren?

Du kannst deinen eigenen Server aufsetzen, der eine Hosts-Datei, mit zu blockierenden Hostnamen zur Verfügung stellt. Dieser muss dann in den Hosts-Quellen hinzugefügt werden. Alternativ können die Hostnamen, die blockiert werden sollen, zur Blacklist unter Deine Listen hinzugefügt werden. Hostnamen, die nicht blockiert werden sollen können zur Whitelist hinzugefügt werden und Hostnamen die zu einer spezifischen IP Adresse umgeleitet werden sollen zur Umleitungsliste.

Wo finde ich mehr Hosts-Quellen?

Auf der Webseite von AdAway gibt es eine Liste zusätzlicher Hosts-Quellen.
