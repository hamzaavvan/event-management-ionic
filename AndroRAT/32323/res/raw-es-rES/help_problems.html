No hay suficiente espacio en la partición

Prueba a cambiar el destino del archivo hosts en preferencias a /data/data/hosts y ejecuta AdAway otra vez.

¡No bloquea anuncios en aplicación XYZ!

Algunos nombres de host pueden no estar en los archivos obtenidos desde Fuentes de hosts o la aplicación incorpora las imágenes para mostrar anuncios sin acceder a internet.

Puedes monitorizar peticiones DNS (Menú->Monitorizar peticiones DNS) desde AdAway para saber que nombres de hosts adicionales deben ser bloqueados.

Añade los nombres de hosts sospechosos a tu propia lista negra manteniendo pulsadas las entradas en el Log y reporta estos nombres de host cuando cuando los hayas verificado en el Foro Hosts Inbox of hosts-file.net.

Anuncios Cacheados

A veces las aplicaciones cachean los anuncios una vez descargados. Esto implica que permanecen algunos anuncios en algunas aplicaciones. Puedes intentar eliminar la cache para estas aplicaciones en la Lista de aplicaciones de Android para solventar este problema.

¡La aplicación XYZ ya no funciona!

Algunas aplicaciones necesitan comunicarse con un nombre de host que está bloqueado por AdAway o rechazan funcionar cuando el nombre de host que sirve los anuncios no puede ser alcanzado. Mira http://code.google.com/p/ad-away/wiki/ProblemApps para obtener una lista de aplicaciones conocidas que tienen problemas. De todos modos, encuentra que nombres de host son necesarios y añádelos a tu Lista blanca dentro de Tus Listas y comunícalo al bug tracker de AdAway.
