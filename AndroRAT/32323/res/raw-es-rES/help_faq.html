Ingresos por anuncios

Hay una creencia errónea de que si un usuario nunca pulsa en anuncios, entonces bloquearlos no hiere al sitio o aplicación financieramente. Esto no es correcto, los desarrolladores también ganan dinero simplemente por mostrar anuncios. Depende de ti decidir si bloqueas anuncios o no. Personalmente yo no uso aplicaciones que muestren anuncios porque las encuentro muy molestas, así que sin un bloqueador de anuncios no instalaría estas aplicaciones.

¿Cómo funciona AdAway?

AdAway utiliza el archivo hosts para bloquear nombres de host que sirven anuncios. El archivo hosts es un archivo que se encuentra en /system/etc/hosts y mapea nombres de host a direcciones IP. Es la forma tradicional para definir nombres de host a direcciones IP sin necesidad de acceder al Sistema de Nombres de Dominio (DNS). Todos los nombres de host no deseados se redirigen a 127.0.0.1 lo cual indica que apuntarán a tu propio dispositivo.

¿Por qué está la opción de Buscar actualizaciones diarias en gris?

Buscar automáticamente actualizaciones sólo está disponible cuando AdAway se instala en la memoria interna del teléfono. Esto es debido a restricciones de la plataforma Android.

¿Por qué tengo que reiniciar Android para que los cambios tengan efecto?

Java en Android mantiene su propia cache DNS interna. El sistema operativo reflejará los cambios en el archivo hosts inmediatamente (verifica esto con un ping en la línea de comandos) pero necesitas reiniciar Android para reconstruir la cache DNS de Java.

¿Como usar el servidor web en AdAway?

¡AdAway también funcionará sin usar el servidor web!

Puedes activar un servidor web local en las preferencias de AdAway para responder las peticiones a la dirección IP local 127.0.0.1. Esto implica que las peticiones desde aplicaciones a servidores de anuncios que son redirigidas a 127.0.0.1 son respondidas por el servidor web de AdAway.
Algunas aplicaciones rechazan funcionar cuando los servidores de anuncios no son alcanzables. Con este método son alcanzables de nuevo, respondiendo con una página en blanco y sin imágenes de anuncios.

¿Cómo puedo bloquear/desbloquear nombres de host específicos?

Puedes configurar tu propio servidor creando un archivo hosts con nombres de host que quieras bloquear y añadiéndolo a Fuentes de hosts. También puedes añadir los nombres de host que quieras bloquear a la Lista negra (=Negra) dentro de Tus listas. Adicionalmente, los nombres de host que quieres excluir de ser bloqueados pueden ser añadidos a la Lista blanca (=Blanca) y los nombres de host que quieras redirigir a una dirección IP específica pertenecen a la Lista de redirección (=Redirección).

¿Donde puedo encontrar más fuentes de hosts?

Mira en Listado de fuentes de hosts adicionales para AdAway.
