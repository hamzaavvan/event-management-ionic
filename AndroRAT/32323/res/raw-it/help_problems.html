Spazio non sufficiente nella partizione

Prova a cambiare il file hosts target nelle preferenze in /data/data/hosts e applica di nuovo AdAway.

Non blocca le pubblicità nell'applicazione XYZ!

Alcuni hostname potrebbero mancare nei file hosts forniti in Sorgenti Host oppure l'applicazione contiene al suo interno le immagini per mostrare pubblicità senza collegarsi a internet.

Puoi registrare le richieste DNS (Menu->Registra Richieste DNS) da AdAway per scoprire quali ulteriori hostname devono essere bloccati.

Aggiungi gli hostname sospetti alla tua personale Blacklist tenendo premuta la voce nel Registro e quando li hai verificati riportali nel Forum Hosts Inbox of hosts-file.net.

Pubblicità conservata nella cache

Alcune applicazioni conservano le pubblicità nella cache dopo il download. Questo fa sì che le pubblicità rimanghino in alcune applicazioni. Puoi provare a eliminare la cache di queste applicazioni nella Lista delle Applicazioni Android per risolvere questo problema.

L'applicazione XYZ non funziona più!

Alcune applicazioni hanno bisogno di comunicare con un hostname bloccato da AdAway oppure si rifiutano di funzionare quando l'hostname che dovrebbe servire a fornire pubblicità non è più raggiungibile. Visita http://code.google.com/p/ad-away/wiki/ProblemApps per ottenere una lista di applicazioni conosciute che hanno problemi. Oppure, scopri quali hostname sono necessari e aggiungili nella Whitelist sotto Le tue liste e riportali sul bug tracker di AdAway.
