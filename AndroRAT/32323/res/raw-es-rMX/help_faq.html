Los ingresos por la publicidad

Hay una idea falsa que se ha dicho en reiteradas ocasiones, que si un usuario nunca hace clic en los anuncios, el sitio o a la aplicación no tendrá beneficios financieros. Esto no es cierto, los desarrolladores también ganan dinero con sólo mostrar anuncios. Depende de si usted bloquea anuncios o no. Personalmente yo no usaría aplicaciones que muestran anuncios porque me parecen muy molestos, así que sin un bloqueador de anuncios yo no instalaría ese tipo de aplicaciones.

¿Cómo funciona AdAway?

AdAway usa el archivo de servidores web para bloquear aquellos que tienen publicidad. Este archivo se encuentra en /system/etc/hosts y registra los servidores en direcciones IP. Es una forma tradicional de definir los servidores web en pares de direcciones IP sin confiar en el sistema de dominio (DNS). Todos los servidores web no deseados son redirigidos a 127.0.0.1, que significa que estos apuntarán a tu propio dispositivo.

¿Por qué la preferencia de comprobación de actualización diaria está sin seleccionar?

La comprobación automática de nuevas actualizaciones sólo está disponible cuando está instalado el AdAway en la memoria interna del teléfono. Esto se debe a la limitación de la plataforma Android.

¿Por qué tengo que reiniciar Android para que los cambios surtan efecto?

Java en Android mantiene su propia caché DNS interno. El sistema operativo mostrará el nuevo archivo de servidores inmediatamente (verificar con ping en la línea de comandos), pero tendrás que reiniciar Android para reconstruir la caché de DNS de Java.

¿Cómo utilizar el servidor web en AdAway?

¡AdAway también trabajará sin utilizar el servidor web!

Puedes habilitar un servidor web local en las preferencias de AdAway para responder a las solicitudes de la dirección local IP 127.0.0.1. Esto significa que las solicitudes de aplicaciones a servidores de anuncios que se redirigen a 127.0.0.1 son contestadas por el servidor web AdAway.
Algunas aplicaciones se niegan a trabajar cuando los servidores de anuncios no son accesibles. Con este método dichas aplicaciones se vuelven accesibles, respondiendo con una página en blanco y no con imágenes publicitarias.

¿Cómo puedo bloquear / desbloquear servidores web específicos?

Usted puede configurar su propio servidor proporcionando un archivo de servidores web con los servidores que desea bloquear, y añadirlo a la Fuente de Servidoress o añadir los anfitriones web que desea bloquear a la lista negra (=Negro) bajo Mis Listas. Además, los servidores web que desea excluir del bloqueo puede ser añadido a la Lista Blanca (=Blanco) y los servidores que desea redirigir a una dirección IP específica que pertenecen a la Lista de Redireccionamiento (=Redirectión).

¿En dónde puedo encontrar más fuentes de servidores web?

Mira en Lista de servidores adicionales para AdAway.
